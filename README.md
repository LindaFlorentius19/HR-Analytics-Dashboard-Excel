# HR-Analytics-Dashboard-Excel
HR Analytics Dashboard built in Microsoft Excel to analyze employee attrition trends across departments, age groups, gender, marital status, and job roles.

## Project Overview
An interactive HR Analytics Dashboard developed in Microsoft Excel to analyze employee attrition and workforce demographics.

## Business Problem
Organizations need to understand attrition trends and identify factors contributing to employee turnover.

## Dashboard Insights
- Employee Count: 440
- Attrition Count: 59
- Attrition Rate: 13%
- Average Age: 37

## Analysis Performed
- Gender-wise Attrition
- Age-wise Attrition
- Department-wise Attrition
- Marital Status-wise Attrition
- Job Role-wise Attrition
